# Core Beliefs

1. Divisibility is required but not sufficient for small `n`.
2. Derivation-based nonexistence propagation is a first-class gate.
3. Strict feasibility invariants (`overcovered == 0`) are more transferable than scalar score.
4. Symmetry/Kramer mode should be bounded and diagnostics-driven.
5. Residual exact-cover should run only after explicit eligibility thresholds.
6. Each active hypothesis must have a falsification condition.
7. Source transfer is valid only when theorem/mechanism and code delta mapping are explicit.
