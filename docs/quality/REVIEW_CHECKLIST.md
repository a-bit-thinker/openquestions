# Review Checklist

## Author Agent
- [ ] Updated canonical docs artifacts for any knowledge change.
- [ ] Added/updated evidence references.
- [ ] Declared falsification criteria for active hypotheses.

## Skeptic Agent
- [ ] Challenged each active claim with at least one counter-risk.
- [ ] Checked source transfer entries for mechanism specificity.
- [ ] Checked derivation-veto assumptions and certificate status labels.

## Verifier Agent
- [ ] Schema validation passes for all structured artifacts.
- [ ] Frontier queue is consistent with instance registry.
- [ ] Gate scorecard updated from current run evidence.
