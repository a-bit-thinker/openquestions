# Derivation Rules

## Closure Rule
If `S(r,q,n)` exists, then `S(r-1,q-1,n-1)` exists (derive by fixing one point).

## Operational Consequence
A verified nonexistence certificate for a base triple can veto all parents reachable by reverse derivation.
