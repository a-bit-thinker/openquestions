# Steiner Domain

A Steiner system `S(r,q,n)` is a set of `q`-subsets of `[n]` such that every `r`-subset is contained in exactly one block.

## Project Window
- `n > q > r > 5`
- `r < 10`
- `n < 200`

## Equivalent Formulation
Exact cover over rows `C([n], r)` and columns `C([n], q)` with exact-once row coverage.
