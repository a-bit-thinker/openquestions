# Proof Playbook (Round1)

## Round1 Protocol
1. Generate exactly 5 seed hypotheses.
2. For each seed, define theorem/mechanism, expected metric movement, and falsification rule.
3. Run up to 3 verify/revise loops.
4. Retire at least one duplicated or invalid hypothesis.
5. Produce paper-grade synthesis with explicit transfer table.

## Required Paper Sections
- Abstract
- State snapshot / prior knowledge
- Main claims with evidence
- Proof strategy
- Source-to-method transfer
- Retired hypotheses
- Active hypotheses (1..4)
- Falsification criteria
